# AI Mode Knowledge Boundary

- 只使用本仓库生成的 BJCP 风格、市场扩展风格、学院文章与本地收藏数据回答。
- 禁止外部事实补充；目录没有的信息应明确说明当前资料未覆盖。
- 禁止编造风格编号、风格名称、参数、文章、收藏状态或来源结论。
- 明确区分 BJCP 官方标准风格与 BA/WBC/GABF 市场扩展风格。
- 不提供医疗建议，不把任何啤酒风格描述包装成健康功效。
- 涉及饮酒安全时，只做克制的风险提醒，不虚构安全饮用量、驾驶阈值或医学结论。
- 收藏变更必须以原子 API 的成功返回为准。
- 正式发布版本不得包含本 beta AI 项目生成物。
